# Creme-Studio 

Cremè Studio is a conceptual bakery and creative baking space designed as a web experience project.

The platform combines:
- Bakery shopping
- Baking sessions
- Recipe exploration
- Interactive cart experience

## Features

- Responsive web design
- Interactive session slider
- Shopping cart using localStorage
- Product quantity system
- Clean and warm UI experience

## Technologies

- HTML
- CSS
- JavaScript
- GitHub Pages
